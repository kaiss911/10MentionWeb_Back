BDD

Les bases de données sont des magasins qui permettent de stocker des informations et permettent notamment d'effectuer différent opération de lecture et d'écriture.

Elles sont efficaces pour retrouver des informations qui y sont stockées. Vous pouvez interroger une base de données, par exemple: "combien d'utilisateurs sont inscrits sur mon site web?" ou "Quel article dans mon blog contiet tel ou tel mot-clé"

La force des bases de données réside également dans leur possibilité de lier les différentes information qui y sont préalablement stockées.

SBDD

SGBD(Systéme de Gestion de Base de Données) permet d'accueilir, d'exploiter et de faire fonctionner les base de données (moteur). Nous utiliserons MySQl, d'autres SGBD existe : Oracle, SQL Server, PostgreSQL, ect.

MySQL

MySQL est un SGBDR (Systéme de Gestion de Bases de Données Relationnelles). Cest l'un des plus utiliser dans le monde du web

Comme sont nom l'indique, MySQL utilise le languague SQL pour effectuer des requetes

Toutes ces actions de gestion et manipulation passeront par une requête SQL

SQL

SQL(Stuctured Query Languague) language de requête structurer : ets un language informatique permettant d'interagire avec des bases de données relationnelles

Une fois que les informations on été enregistrées, il est important de pouvoir les gérer, ajouter, modification, suppression, consultation

phpMyAdmin

phpMyAdmin(souvent abrégé PMA) est une application web de gestion pour MySQL. Il est ecrit en PHP et permet de gérer les BDD dans une interface visuelle simple et intuitive

Il est important de noter que phpMyAdmin n'est qu'une application. Il est tout a fait possible de gérer MySQL avec d'autres applications similaires ou en ligne de commande

CRUD

CRUD(Create, Read, Update, Delete) désigne les quatres opérations de base pour le stockage et la manipulation d'informations dans une base de données

  • Create: Insérer des nouveaux enregistrements (écriture)
  • Read: Récupérer des enregistrements (lecture)
  • Update: Mettre à jour des enregistrements (écriture)
  • Delete: Supprimer des enregistrements (écriture)

Les opérations correspondent chacun à un type de requête SQL
