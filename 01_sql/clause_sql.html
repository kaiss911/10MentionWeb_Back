1 - Les requêtes de bases

  • Créer une BDD : CREATE DATABASE IF NOT EXISTS nomBdd
  • Supprimer une BDD : DROP DATABASE IF EXISTS nomBdd
  • Observer la description d'une table : DESC nomBdd
  • Voir les BDD dans le serveur : SHOW DATABASES

2 - Tables et colonnes

  • Créer une BDD 'entreprise' : CREATE DATABASE IF NOT EXISTS entreprise
  • Créer une table dans la BDD entreprise : 
    
                                CREATE TABLE IF NOT EXISTS employes(
                                id_employes INT(11) NOT NULL PRIMARY KEY AUTO_INCREMENT,
                                prenom VARCHAR(20) NULL,
                                nom VARCHAR(20) NULL,
                                sexe ENUM ('f', 'm') NOT NULL,
                                service VARCHAR(30) NULL,
                                date_embauche DATE NULL,
                                salaire FLOAT NULL
                                )

3 - select et distinct

  • Pour voir tout le contenue d'une table : SELECT * FROM employes(* = tout = all)
  • Sélectionner les valeurs de deux champs d'une table : SELECT colonne1, colonne2 FROM nomTable -> SELECT prenom, nom FROM employes
  • Afficher les données sans doublons : SELECT DISTINCT colonne FROM nomTable -> SELECT DISTINCT service FROM employes
  • Effacer un enregistrement: DELETE FROM nomTable WHERE colonne ='' -> DELETE FROM employes WHERE nom = 'Salimata'
